Renaming and Conversion Tool for Epic Mickey

Please read the following before using the program:

Requirements

  • Renaming and Conversion Tool for Epic Mickey (this program)
  • Ravioli Game Tools (version 2.10 or newer)
  • Epic Mickey extracted game files (how?)
  • A computer with Windows 7 or newer
  • ~4GB of free space for audio files

Notes

  • The amount of time that the process will take to run will depend on your computer's speed and available resources. It may take up to 10 minutes.
  • Your CPU usage will spike while the process is running, possibly causing your PC to become slower until it is complete. I recommend grabbing a snack or finding something else to do while the program is working.
  • Some files cannot be renamed due to there being no name in SoundbanksInfo.xml. These files are placed in __Converted\Unnamed\. This is not a bug.
  • If the program closes unexpectedly or you encounter an error, check __Converted\log.txt.
  • Check __Converted\skipped_files.txt after the process is complete to make sure the program didn't have to skip any files.
  • Please report any errors to me through one of the contact links on my site.

Instructions

    Setting up files

  1. Create a working folder somewhere on your PC.
  2. Navigate to your Epic Mickey extracted game files and go to Audio\Wii\. Select all of the files in the folder (ctrl+a) and copy them to the clipboard. Then, paste them into your working folder.
  3. Close the Epic Mickey game files and go back to your working folder. You should see a Localize folder. This contains different language-specific audio files. Copy any of the files you would like to process into the root of your working folder. Files in Localize will not be processed unless they are moved to the root.

    4. Ravioli Game Tools

  4. Navigate to wherever you installed Ravioli Game Tools and run RExtractor.exe.
  5. Click the Browse... button next to Input file(s) and locate your working folder. Select all of the bnk files in this folder and click Open.
  6. Now click the Browse... button next to Output directory and set it to your working folder.
  7. Click the Start button. Once it is complete, you may close Ravioli Game Tools.

    8. Running the Renaming and Conversion Tool

  8. Unzip the Renaming and Conversion Tool into your working folder. (EMAudioProcess.zip)
  9. Go to your working folder and run _AudioProcess.exe. Type "start" in the window that pops up.
  10. The program will now rename and process all of the wav, wem, and ogg files in your working folder. Once it is complete, the files can be found in the __Completed folder.

Program information

This program was written in Python 3.7 by Andrew Cole. The (very messy and unreadable) source code can be found in _rename_and_convert.py.txt.
